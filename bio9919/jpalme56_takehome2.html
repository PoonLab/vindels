Generate a visual confirmation of this with a scatterplot that displays this ratio on the x-axis, and RA on the y-axis:

  1. Make a legend for your barplot and display it along the top. Remember to use levels to extract a character vector for Phylum. Since this legend is going to be too large for the plot region, youre going to have to tinker with the plot settings with par. Here are some tips:
  • par(xpd=NA) allows you to plot objects outside the normal region (i.e., in the margins)
  • par(margin=c(5,5,1,5)) creates extra space on the right side (fourth value).
  • legend(..., cex=0.5) is a useful way to make the entire legend smaller.
  • legend(..., fill=<vector>) can be used to draw filled boxes with different colours contained in <vector>.
